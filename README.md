# ks_autotest         作者:kangs2008

## 版本：V 2.0
- API测试有俩种方式进行
- 1,通过excel来编写用例，以excel来驱动测试框架，不用写python脚本
- 2,通过写python脚本的方式，进行测试
- WEB UI 测试：极简代码输出，减少python脚本编写工作量。
- log报告文件输出
- 错误自动截图和可按需截图和截图标记

## 项目说明
- 本框架是基于**pytest+allure/html**设计

## 技术栈
- requests
- pytest
- pytest-html
- pytest-allure
- openpyxl
- logging
- cv2
